### 2. 测试基础接口

使用您的测试代码验证接口：
```javascript
const axios = require('axios');

const BASE_URL = 'http://localhost:8080';
const ADMIN_TOKEN = 'your_admin_token_here'; // 请替换为实际的管理员token

const headers = {
    'Authorization': `Bearer ${ADMIN_TOKEN}`,
    'Content-Type': 'application/json'
};

// 测试基础接口
async function testBasicAPI() {
    try {
        const response = await axios.get(`${BASE_URL}/admin/like-favorite/stats`, { headers });
        console.log('接口正常:', response.data);
    } catch (error) {
        console.error('接口错误:', error.response?.status, error.response?.data);
    }
}

```

### 4. 常见问题解决

#### 问题1：404错误
**原因**：接口路径不正确或后端未实现
**解决**：
- 确认后端接口路径
- 检查后端Controller映射
- 验证请求方法（GET/POST/DELETE）

#### 问题2：401/403错误
**原因**：权限验证失败
**解决**：
- 检查Token是否有效
- 确认用户有管理员权限
- 验证Authorization头格式

#### 问题3：跨域错误
**原因**：CORS配置问题
**解决**：
- 检查后端CORS配置
- 确认允许的Origin
- 验证请求头设置
